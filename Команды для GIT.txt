cd - открыть папку
git clone http...... - клонирование репазитория
git status — команда получения статуса файлов в репозитории.
git add . - чтобы файл попал под версионный контроль и впоследствии в нём можно было зафиксировать изменения
git commit - Для фиксации изменений необходимо выполнить команду 
git push - команда, которую нужно использовать для публикации изменений на сервере
git log - командa, показывающей историю изменений
git rm - команда на удаление
git branch -Для создания новой ветки в Git
git branch --list  - выведет список веток, которые существуют в вашем локальном репозитории
git branch --list -v - Вы увидите информацию о последних коммитах каждой из веток
git checkout - необходимо выполнить для переключения на созданную ветку (Например, git checkout testbranch00)
